# Wayne Lin Resume — Overleaf 使用说明

## 在 Overleaf 中使用

1. 打开 [Overleaf](https://www.overleaf.com) → **New Project** → **Upload Project**
2. 上传 **Wayne-Lin-Resume.tex**（可单独上传此文件）
3. 菜单 **Menu** → **Compiler** 选择 **pdfLaTeX**
4. 点击 **Recompile** 生成 PDF

## 用 Word 简历内容替换占位符

- **姓名 / 联系方式**：替换 `\name{Wayne Lin}` 和 `\contactline{...}` 中的邮箱、电话、LinkedIn、城市
- **Education**：替换学校、专业、毕业时间、GPA、相关课程
- **Experience**：每条经历保持格式  
  **职位** \hfill **时间**  
  _公司名_ \hfill 城市
  - 动词开头 + 做了什么 + **数字/百分比结果**
- **Projects**：项目名、技术栈、你的角色与成果
- **Skills**：只保留与目标岗位相关的技能，可分类（语言/工具、领域、其他）

## 优秀简历要素（本模板已遵循）

- 结构清晰、一页为主
- 经历用 **STAR** 思路：情境/任务 → 行动 → 结果（尽量量化）
- 动词开头：Led, Delivered, Built, Improved, Reduced, Collaborated
- 真实、可核实，不夸大
