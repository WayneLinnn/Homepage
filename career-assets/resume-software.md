# Software Resume Focus

## Target roles

- Java Backend Engineer
- Backend-leaning Full-Stack Engineer
- Software Engineer
- Application Engineer

## What to lead with

- Java, Spring Boot, API delivery
- Backend services, integration, production support
- Business workflow understanding
- Collaboration with frontend and product teams

## What to de-emphasize

- Pure research framing
- Generic "AI expert" wording
- Too many unrelated tools

## Bullet-writing pattern

- Built X for Y team, resulting in Z outcome
- Delivered API or workflow integration for production use
- Improved reliability, speed, or clarity of a business process

## Suggested summary line

PR-based backend / full-stack engineer in Australia with strong Java and Spring Boot delivery experience, focused on shipping reliable business systems and growing AI-enabled product capability.
