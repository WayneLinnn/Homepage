# Learning Roadmap

## Layer 1: Software + AI foundations

- React
- TypeScript
- Python
- Machine learning basics
- Docker

## Layer 2: Product and deployment

- REST APIs
- Auth and permissions
- Cloud deployment
- MLflow
- Evaluation workflows

## Layer 3: Platform and MLOps

- Kubernetes fundamentals
- Model serving
- Monitoring
- Kubeflow or KServe awareness

## Recommended order

1. Strengthen React and TypeScript enough to ship full-stack features
2. Build one Python AI application with a clear API and UI boundary
3. Containerize both projects with Docker
4. Learn cloud deployment basics
5. Add evaluation, monitoring, and then Kubernetes awareness
